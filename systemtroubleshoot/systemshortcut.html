Amazing Computer Keyboard Shortcuts

Computer keyboard shortcuts are quick key combinations that help users perform tasks faster without using the mouse. Using keyboard shortcuts regularly saves time, improves productivity, and makes computer operation more professional and convenient.

Open Run, Windows Button + R

Open Notepad, Go to Run, then Type notepad

Open Command Prompt, Go to Run, then Type cmd

Open MS Word, Go to Run, then Type winword

Open MS Excel, Go to Run, then Type excel

Open MS Power Point, Go to Run, then Type powerpnt

Open MS paint, Go to Run, then Type mspaint

Open Chrome Browser, Go to Run, then Type chrome

Open Firefox Browser, Go to Run, then Type firefox

Open Edge Browser, Go to Run, then Type msedge

Open Internet Explorer, Go to Run, then Type iexplore

Open Task Manager, Press Ctrl + Shift + Esc

Open Disk Management, Go to Run, then Type diskmgmt.msc

Open Device Management, Go to Run, then Type devmgmt.msc

Open Certificate Manager, Go to Run, then Type certmgr.msc

Open MS Power Point, Go to Run, then Type powerpnt

Open LAN Settings, Go to Run, then Type ncpa.cpl

Open Programs and Features (Unistall Softwares), Go to Run, then Type appwiz.cpl

Restart the System, Go to Run, then Type shutdown -r -t 0

Shutdown the System, Go to Run, then Type shutdown -s -t 0
