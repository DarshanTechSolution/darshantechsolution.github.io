Amazing Computer Keyboard Shortcuts

Computer keyboard shortcuts are quick key combinations that help users perform tasks faster without using the mouse. Using keyboard shortcuts regularly saves time, improves productivity, and makes computer operation more professional and convenient.

1. Open Run, Windows Button + R

2. Open Notepad, Go to Run , then Type notepad

3. Open Command Prompt, Go to Run, then Type cmd

4. Open MS Word, Go to Run, then Type winword

5. Open MS Excel, Go to Run, then Type excel

6. Open MS Power Point, Go to Run, then Type powerpnt

7. Open MS paint, Go to Run, then Type mspaint

8. Open Chrome Browser, Go to Run, then Type chrome

9. Open Firefox Browser, Go to Run, then Type firefox

10. Open Edge Browser, Go to Run, then Type msedge

11. Open Internet Explorer, Go to Run, then Type iexplore

12. Open Task Manager, Press Ctrl + Shift + Esc

13. Open Disk Management, Go to Run, then Type diskmgmt.msc

14. Open Device Management, Go to Run, then Type devmgmt.msc

15. Open Certificate Manager, Go to Run, then Type certmgr.msc

16. Open MS Power Point, Go to Run, then Type powerpnt

17. Open LAN Settings, Go to Run, then Type ncpa.cpl

18. Open Programs and Features (Unistall Softwares), Go to Run, then Type appwiz.cpl

19. Restart the System, Go to Run, then Type shutdown -r -t 0

20. Shutdown the System, Go to Run, then Type shutdown -s -t 0
